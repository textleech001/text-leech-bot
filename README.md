## Deploy To VPS
```
git clone https://github.com/textleech001/text-leech-bot
```

```
cd text-leech-bot
```

```
nano modules/vars.py
```

USE : `ctrl + s` : FOR SAVE | `ctrl + x` : FOR EXIT

```
sudo apt install docker.io -y
```

```
sudo docker build -t text-leech-bot-by-king .
```

```
sudo docker run -it --rm text-leech-bot-by-king
```
